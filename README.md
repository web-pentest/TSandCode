# ⚡ TSandCode — песочница кода

**Живой редактор HTML/CSS/JS в браузере. Пиши код — смотри результат мгновенно.**

[![Live Demo](https://img.shields.io/badge/demo-online-brightgreen)](https://web-pentest.github.io/TSandCode/)
[![GitHub license](https://img.shields.io/badge/license-MIT-blue)](LICENSE)
[![Made with](https://img.shields.io/badge/made%20with-❤️-red)](https://github.com/web-pentest)

---

## ✨ Возможности

- 📝 Три панели: HTML, CSS, JavaScript
- 👁️ Живой вывод результатов
- 💾 Автосохранение в браузере (localStorage)
- 🌙 Тёмная тема в стиле киберпанк
- 📱 Адаптивный дизайн (работает на телефонах)

---

## 🚀 Быстрый старт

1. Перейди на [демо-страницу](https://web-pentest.github.io/TSandCode/)
2. Пиши код в левых панелях
3. Результат появляется справа автоматически

---

## 🛠️ Технологии

- HTML5
- CSS3
- JavaScript (ES6)

---

## 📁 Локальный запуск

```bash
git clone https://github.com/web-pentest/TSandCode.git
cd TSandCode
# просто открой index.html в браузере
```
